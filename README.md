
# E-Commerce App using Strapi

This is an e-commerce shopping app for users to add to cart, remove items 
from cart, delete items and proceed to checkout.




## API Reference

#### Get all products

```http
  GET /products?populate=*
```


#### Get product

```http
  GET /products/${product.id}
```
